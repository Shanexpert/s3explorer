# The Jenkins plugin for working with an S3 bucket.

## Build
```
mvn install
```

## Install
Go to _Manage Jenkins_ -> _Manage Plugins_ -> _Advanced_ -> _Upload Plugin_  
Upload file `s3explorer/target/com.finc.s3explorer.hpi`
